# API Service for Too Lazy To Watch

## Stack
* Language: Golang
* Database: Supabase
* Storage: Supabase
* Authentication: Supabase
* Queue: RabbitMQ

# How to Install
* `copy .env.example .env`
* `go mod install`

# How to Run in Localhost
* Ensure you use the latest `docker-compose`
* `make run-dev`
